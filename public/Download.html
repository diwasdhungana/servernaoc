How to install instruction

  1. Download client side from out Github page.
  2. Install Nodejs in your PC
  3. Do npm rebuild and install in the client side downloaded file.
  4. You can find the working page in your browser @ (localhost:3...)
  5. your specific localhost port can be found in the CMD/terminal. check to see where it's running.
  6. To send coins, Navigate to transactions and enter recepient address and amount you're willing to send.
  7. Consider looking at this server site if you're feeling difficulty with anything.
