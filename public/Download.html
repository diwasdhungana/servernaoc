Client Source Code

This code allows users to carry out transactions between other users.Through this code, a user can genereate his/her public key for carrying out transactions between other users.




Server Source Code

This server code notifies the user about how the transactions are being processed or carried out transparently. Through this, a user can see how his transactions are being processed. This server code informs about how the blocks are being mined, how transactions are being processed.

How to install instruction

  • Download client side from out Github page.
  • Install Nodejs in your PC
  • Do npm rebuild and install in the client side downloaded file.
  • You can find the working page in your browser @ (localhost:3...)
  • your specific localhost port can be found in the CMD/terminal. check to see where it's running.
  • To send coins, Navigate to transactions and enter recepient address and amount you're willing to send.
  • Consider looking at this server site if you're feeling difficulty with anything.

Switching Accounts

If a user wants to transfer his/her coins/account from one computer to another, He/She shall only copy the 'private.pem' and 'public.pem' files available in the keys folder to another computer. (Do note: The another PC must have succesful setup of NAO coin project.)
